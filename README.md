# Hostile Mob Better Configuration

Hostile Mob Better Configuration is a highly advanced configuration mod that allows complete control over hostile mobs in Minecraft. It provides a detailed configuration system that lets you modify mob attributes, combat behavior, spawning mechanics, special abilities, and boss mechanics.

Every supported mob can be individually customized using a clean and organized configuration interface. Players and modpack creators can rebalance the difficulty of the game, create powerful boss encounters, or completely redesign mob behavior.

The configuration is divided into multiple categories including Overworld mobs, Nether mobs, End mobs, and major boss mobs.

---

# Core Features

This mod allows full customization of hostile mobs through a large set of configurable properties.

General configuration features include:

Custom mob names
Health and armor adjustments
Attack damage and knockback configuration
Movement speed and follow range
Spawn rate multipliers
Reinforcement summoning chances
Experience drop multipliers
Loot drop multipliers
Fire immunity options
Glowing effects
Door breaking behavior

Each mob can be tuned individually to create unique gameplay experiences.
