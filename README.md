<h1 style="text-align: center;"><strong><span style="color: #f1c40f;">Hostile Mob Better Configuration</span></strong></h1>
<p>Hostile Mob Better Configuration is a highly advanced configuration mod that allows complete control over All hostile mobs in Minecraft. It provides a detailed configuration system that lets you modify mob attributes, combat behavior, spawning mechanics, special abilities, and boss mechanics.</p>
<p>Every supported mob can be individually customized using a clean and organized configuration interface. Players and modpack creators can rebalance the difficulty of the game, create powerful boss encounters, or completely redesign mob behavior.</p>
<p>The configuration is divided into multiple categories including Overworld mobs, Nether mobs, End mobs, and major boss mobs.</p>
<hr>
<h1 style="text-align: center;"><span style="color: #f1c40f;">Core Features</span></h1>
<p>This mod allows full customization of hostile mobs through a large set of configurable properties.</p>
<p>General configuration features include:</p>
<p>Custom mob names<br>Health and armor adjustments<br>Attack damage and knockback configuration<br>Movement speed and follow range<br>Spawn rate multipliers<br>Reinforcement summoning chances<br>Experience drop multipliers<br>Loot drop multipliers<br>Fire immunity options<br>Glowing effects<br>Door breaking behavior<br>Custom combat mechanics<br>Boss mode system</p>
